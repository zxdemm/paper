# Automatic Software Repair: A Bibliography

自动修复的定义：根据规范(specifications)将程序执行中不可接受的行为转换为可接受的行为

一般来说，一种修复技术只针对一种bug类，即相同的根本原因、相同的症状和相同的解决方法
